# 📚 Software Testing Notes

A clean and structured study notes website for **Software Testing** — built with [VitePress](https://vitepress.dev/) and deployed on GitHub Pages.

## 🌐 Live Site

👉 [kuntalojha.github.io/st](https://kuntalojha.github.io/st/)

## 📖 What's Inside

- **Unit 1** — Basics of Software Testing (SDLC, STLC, V-Model, Agile, Test Cases)
- **Unit 2** — Types & Levels of Testing (Unit, Integration, System, Acceptance, Performance)
- **Unit 3** — Testing Methods (White Box, Black Box, BVA, Equivalence Partitioning)
- **Unit 4** — Test & Defect Management (Test Planning, Defect Life Cycle, Reporting)

## 🚀 Run Locally

```bash
git clone https://github.com/kuntalojha/st.git
cd st
npm install
npm run docs:dev
```

## 🛠️ Built With

- [VitePress](https://vitepress.dev/) — Static site generator
- GitHub Pages — Hosting
