# Software Testing

## Unit 1

- **Basic of software testing**Software Testing, Objective of Testing, Failure, Error, Fault, Defect, Bug Terminology. Sdlc, Stlc, Waterfall Model, Verification and validation (V Model), Agile Model, Test case, When to Start and Stop Testing of Software (entry and exit criteria). 

## Unit 2

- **Types and Levels of Testing** – Levels Of testing, Unit Testing: Unit test Planning, Designing, Running the unit test and Recording result, Integration testing: Integration test planning, Designing, System testing : test planning, Designing, Acceptance Testing : Alpha Testing and Beta Testing, Testing on web application: Performance testing, stress testing, Security testing, Special Test: Regression Testing and GUI Testing.

## Unit 3

- **Testing methods** - Methods of testing: static and dynamic testing, the box approach: White Box testing: Inspections, Walkthroughs, Technical Reviews, Functional Testing, Black Box Testing: Requirement Based Testing, Boundary Value Analysis, Equivalence Partitioning. Special Test: Regression Testing and GUI Testing.

## Unit 4

- **Test Management & Defect Management** - Test Planning: Preparing a Test Plan, Deciding Test Approach, Setting Up Criteria for Testing, Identifying Responsibilities, Staffing, Resource Requirements, Test Deliverables, Testing Tasks, Defect Classification, Defect Management Process, Defect Life Cycle, and Techniques for Finding Defects, Defect Template, Reporting a Defect.
